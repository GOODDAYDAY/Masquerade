I built an open-source project where AI agents play social deduction games — completely on their own.

The system combines **deterministic orchestration** (engine-driven game rules, turn order, phase transitions) with **agentic reasoning** (LangGraph state graphs for individual cognition: introspection → evaluation → strategy optimization) and **emergent multi-agent dynamics** — no scripted dialogues or hardcoded strategies. Deception, alliances, and betrayal arise naturally from each agent's independent inference, private memory, and role-specific objectives.

Currently supports **Who Is The Spy** and **Werewolf** (with 6 roles: Seer, Witch, Hunter, Guard, Werewolf, Villager).

The best part? Every game automatically renders into a **2K 60fps cinematic video** with AI-generated voice acting — powered by Remotion's frame-driven rendering engine. Deterministic output, perfect audio sync, zero manual editing.

Tech stack: Python + LangGraph + React + Remotion + edge-TTS

The full pipeline — game simulation, voice generation, video rendering — runs with a single command.
